# Inbox: vertical centering + expand/collapse animation

*2026-06-12T19:32:08.911Z*

![Inbox centering and open/close animation](inbox-centering-animation-video-1.gif)

On the landing page, the capture box is now centered vertically in the viewport. When 'View inbox' is clicked, the capture box slides up to its normal position as the inbox list expands below. Clicking 'Close' reverses: the inbox collapses and the capture box slides back to center. Both transitions run at 300ms ease-out.
